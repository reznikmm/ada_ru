       Краткое описание структуры ведения содержимого
                  сайта www.ada-ru.org
                  
Данная структура служит следующим целям:
- ведение и автоматическое построение сайта www.ada-ru.org
- в том числе, ведение каталога свободно распространяемого
  ПО на языке Ада
- построение образа CD-ROM с последними версиями ПО и
  последней версией сайта.
  
Структура каталогов:
bin/   - скрипты и исполняемуе программы
build/ - временные файлы при построении
cdrom/ - место построения образа CDROM
cont/  - исходники сайта
links/ - данные ведения каталога ПО
src/   - Ада тексты сайта и загрузчика

Каталоги не попавшие в CVS :-/
autoexec/ - стартер CD-ROM под Windows
misc/     - бинарные файлы, картинки и архивы для сайта
out/      - место для результатов построения сайта
ref/      - кеш файлов уже лежащих на сайте
store/    - архив ПО для Ады
